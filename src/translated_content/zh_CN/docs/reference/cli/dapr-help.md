### 说明

Help provides help for any command in the application.

### Usage

```bash
dapr help [command] [flags]
```

### Flags

| Name           | Environment Variable | Default | 说明                       |
| -------------- | -------------------- | ------- | ------------------------ |
| `--help`, `-h` |                      |         | Prints this help message |
