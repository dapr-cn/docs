### 描述

Help 命令为应用程序中的任何命令提供帮助。

### 用法

```bash
dapr help [command] [flags]
```

### 标志

| 名称           | 环境变量             | 默认值 | 描述                      |
| -------------- | -------------------- | ------- | ------------------------ |
| `--help`, `-h` |                      |         | 显示此帮助信息           |
