### 描述

获取 `dapr` 和 `daprd` 可执行文件的版本和 Git 提交信息。

### 支持的平台

- [本地托管]({{< ref self-hosted >}})

### 用法

```bash
dapr build-info
```

### 相关信息

您可以通过运行 `daprd --build-info` 命令直接获取 `daprd` 的构建信息。
