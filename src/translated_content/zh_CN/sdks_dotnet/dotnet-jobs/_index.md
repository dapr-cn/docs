使用 Dapr Job 包，您可以在 .NET 应用程序中与 Dapr Job API 进行交互。通过预设的计划，您可以安排未来的操作，并可选择附带数据。

要开始使用，请查看 [Dapr Jobs]({{< ref dotnet-jobs-howto.md >}}) 指南，并参考 [最佳实践文档]({{< ref dotnet-jobsclient-usage.md >}}) 以获取更多指导。
