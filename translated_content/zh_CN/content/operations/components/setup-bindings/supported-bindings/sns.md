## 配置

To setup AWS SNS binding create a component of type `bindings.aws.sns`. 请参阅[本指南]({{< ref "howto-bindings.md#1-create-a-binding" >}})，了解如何创建和应用绑定配置。

关于身份验证相关属性的信息，请参阅 [认证到 AWS]({{< ref authenticating-aws.md >}})

```yaml
apiVersion: dapr.io/v1alpha1
kind: Component
metadata:
  name: <NAME>
  namespace: <NAMESPACE>
spec:
  type: bindings.aws.sns
  version: v1
  metadata:
  - name: topicArn
    value: mytopic
  - name: region
    value: us-west-2
  - name: accessKey
    value: *****************
  - name: secretKey
    value: *****************
  - name: sessionToken
    value: *****************

```

{{% alert title="Warning" color="warning" %}}
以上示例将密钥明文存储。 更推荐的方式是使用 Secret 组件， [这里]({{< ref component-secrets.md >}})。
{{% /alert %}}

## 元数据字段规范

| 字段           | 必填 | 绑定支持 | 详情                      | 示例                  |
| ------------ |:--:| ---- | ----------------------- | ------------------- |
| topicArn     | 是  | 输出   | The SNS topic name      | `"arn:::topicarn"`  |
| region       | 是  | 输出   | The specific AWS region | `"us-east-1"`       |
| accessKey    | 是  | 输出   | 要访问此资源的 AWS 访问密钥        | `"key"`             |
| secretKey    | 是  | 输出   | 要访问此资源的 AWS 密钥访问 Key    | `"secretAccessKey"` |
| sessionToken | N  | 输出   | 要使用的 AWS 会话令牌           | `"sessionToken"`    |
