## 描述

帮助为应用程序中的任何命令提供帮助。

## 用法

```bash
dapr help [command] [flags]
```

## 参数

| Name           | 环境变量 | 默认值 | 描述      |
| -------------- | ---- | --- | ------- |
| `--help`, `-h` |      |     | 显示此帮助消息 |
