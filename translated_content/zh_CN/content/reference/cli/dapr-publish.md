## 描述

发布 pub-sub 事件。

## 支持的平台

- [自托管]({{< ref self-hosted >}})

## 用法

```bash
dapr publish [flags]
```

## 参数

| 名称                 | 环境变量 | 默认值 | 描述                |
| ------------------ | ---- | --- | ----------------- |
| `--publish-app-id` | `-i` |     | 代表您要发布的应用程序的 ID   |
| `--pubsub`         | `-p` |     | Pub/sub 组件的名称     |
| `--topic`, `-t`    |      |     | 待发布的 topic        |
| `--data`, `-d`     |      |     | JSON 序列化数据字符串（可选） |
| `--help`, `-h`     |      |     | 显示此帮助消息           |


## 示例

### 将示例 topic 发布到目标 pubsub
```bash
dapr publish --publish-app-id appId --topic sample --pubsub target --data '{"key":"value"}'
```
